Style

#block { height: 40px; margin: 40px; padding: 40px; border: 40px solid #777; }

Получаем высоту

Лишь значение height

$('#block').height()

Значение height + padding

$('#block').innerHeight()

Значение height + padding + border

$('#block').outerHeight()

Значение height + padding + border + margin

$('#block').outerHeight(true)

Поменяем свойство box-sizing

$('#block') .toggleClass("content-box border-box")
