// обработчики событий для box1 $('#box1') .mousedown('counter') .mouseup('counter') .click('counter') .dblclick('counter') // обработчики событий для box2 $('#box2') .mouseenter('counter') .mouseleave('counter') // обработчики событий для box3 $('#box3') .mousemove('counter') // обработчики событий для box4 $('#box4') .mouseout('counter') .mouseover('counter')

Events:

  • 0mousedown - нажатие клавиши мыши
  • 0mouseup - отжатие клавиши мыши
  • 0click - клик по элементу
  • 0dblclick - двойной клик по элементу
  • 0mousemove - движение курсора
  • 0mouseenter - наведение курсора на элемент, не учитывает на дочерние элементы
  • 0mouseleave - вывод курсора из элемента, не учитывает дочерние элементы
  • 0mouseout - вывод курсора из элемента
  • 0mouseover - наведение курсора на элемент
